# Lily Pad Lane

Lily Pad Lane is a 2D Unity game inspired by classic Frogger-style mechanics. The player navigates across lanes of moving obstacles and platforms while avoiding hazards.

## Demo
Gameplay video: https://youtu.be/qtfJiBoMulk

## Technical Highlights
- Grid-based player movement
- Dynamic row and obstacle generation
- Collision handling with water, logs, and hazards
- Difficulty scaling based on player progress
