# Flappy Bird Clone

A 2D arcade-style game developed in Unity using C#. This project recreates the core Flappy Bird mechanics with a focus on clean game logic, collision handling, and responsive player input.

## Demo
Gameplay video: https://youtu.be/vgRtHdn1_9Q

## Technical Highlights
- Player-controlled vertical movement and physics-based motion
- Procedural obstacle spawning
- Collision detection and scoring system
- Game state management (start, game over, restart)
